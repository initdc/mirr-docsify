# npm registry

## upstream

`registry.npmjs.org`

## official mirror

none

## our mirror

`npm.fastly.mirr.one`

`npm.gcdn.mirr.one`

## usage

```sh
# set
npm config set registry https://npm.fastly.mirr.one

yarn config set registry https://npm.fastly.mirr.one


# unset
npm config set registry https://registry.npmjs.org

yarn config set registry https://registry.npmjs.org
```
