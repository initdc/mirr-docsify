# npm 注册表

## 上游

`registry.npmjs.org`

## 官方镜像

没有

## 我们的镜像

`npm.fastly.mirr.one`

`npm.gcdn.mirr.one`

## 用法

```sh
# set
npm config set registry https://npm.fastly.mirr.one

yarn config set registry https://npm.fastly.mirr.one


# unset
npm config set registry https://registry.npmjs.org

yarn config set registry https://registry.npmjs.org
```
